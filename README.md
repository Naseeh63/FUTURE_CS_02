## Objective
The goal of this task is to analyze phishing email samples, identify phishing indicators, and classify emails based on risk level.

## Tools Used
- Browser inspection tools
- Email analysis techniques
- Public phishing email datasets

## Email Analysis Results

Email 1 – Phishing (High Risk)  
Indicators: Fake domain, urgent message, suspicious link

Email 2 – Suspicious (Medium Risk)  
Indicators: Unknown sender domain, suspicious attachment

Email 3 – Safe (Low Risk)  
Indicators: Legitimate sender, no suspicious link
